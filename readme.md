# SQL Employee Management System (HRMS)

A professional SQL-based Human Resource Management System (HRMS) project built using MySQL.

## Features

- Department Management
- Employee Management
- Project Management
- Employee Project Assignment
- Attendance Management
- Salary Management
- Leave Management

## Database Tables

- Departments
- Employees
- Projects
- Employee_Projects
- Attendance
- Salaries
- Leaves
